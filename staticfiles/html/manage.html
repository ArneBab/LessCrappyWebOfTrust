Manage twitter @users or #hashtags to repeat

All the messages from the twitter user will be repeated by the selected Sone identity. This won't include the name of the twitter handle for tweets that you yourself create on Twitter. Tweets for a search requset (hashtag or regular string) will be posted by the selected Sone identity and will all be prefixed with the twitter handle of the original message.

Add either @username or #hashtag as a pattern in the box.
